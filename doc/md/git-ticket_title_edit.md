## git-ticket title edit

Edit a title of a ticket.

### Synopsis

Edit a title of a ticket.

```
git-ticket title edit [<id>] [flags]
```

### Options

```
  -t, --title string   Provide a title to describe the issue
  -h, --help           help for edit
```
